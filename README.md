# 실습3 (팀 과제)
### 기능
#### A. 미니 도구 모음(Utility Pack) 제작
여러 개의 간단한 기능을 하나의 프로그램으로 묶어 사용자가 메뉴에서 선택하여 사용할 수 있는 **도구 모음 프로그램**
- 단위 변환기(cm ↔ m)
- 간단 계산기(사칙연산)
- 점수 평균 계산
- 문자열 처리(대문자/소문자 변환)
- 랜덤 비밀번호 생성기

---
### 팀 역할 분담
- 이민우 - 단위 변환기(cm ↔ m)
- 정웅현 - 랜덤 비밀번호 생성기
- 강나연 - 간단 계산기(사칙연산)
- 김주혁 - 점수 평균 계산
- 엄서준 - 문자열 처리(대문자/소문자 변환)

---
### 실행 방법
사용자가 메뉴에서 원하는 도구를 키보드 숫자 입력으로 선택하면 그 도구의 기능을 실행할 수 있다.
